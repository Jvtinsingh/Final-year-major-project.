# AcadeMetrics

A unified educational evaluation platform built with [Next.js](https://nextjs.org), Firebase, and modern web technologies.

## Features

- **Multi-Role Access** — Admin, Faculty, and Student dashboards
- **Exam Management** — Create, assign, and take exams (MCQ & Coding)
- **Secure Assessments** — Full-screen lockdown browser environment
- **Real-Time Analytics** — Performance tracking and data visualizations
- **Data Export** — Download institutional data as `.xlsx` workbooks
- **Mock Mode** — Run the entire platform without Firebase credentials

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Database:** Firebase / Firestore
- **Auth:** Firebase Authentication
- **Animations:** Framer Motion
- **Charts:** Chart.js + react-chartjs-2
- **Code Editor:** Monaco Editor

## Deployment

Deployed on [Vercel](https://vercel.com).
